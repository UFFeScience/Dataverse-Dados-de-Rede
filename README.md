# Projeto Dataverse Dados de Rede

> Ferramenta / scripts para ingestão / processamento de dados de rede via Dataverse.  

## Visão geral

O repositório **Dataverse-Dados-de-Rede** tem como objetivo facilitar a carga, organização e disponibilização de dados de rede e de Aprendizado de Máquina (estrutura de rede, topologia, metadados associados etc.) em repositórios baseados em **Dataverse**.  
Este repositório reúne scripts que fazem desde a preparação/transformação dos dados até a importação no Dataverse, visando padronizar e automatizar o processo de integração de dados de rede.

## Funcionalidades principais

- Scripts de carga base  
- Integração / importação no Dataverse via workflow automatizado  
- Possibilidade de reutilização para diferentes estudos ou datasets
